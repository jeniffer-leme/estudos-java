## Exercício 20 – Livro

Implementação da classe **Livro**, conforme o modelo apresentado no exercício 1.19.

### Descrição
A classe representa um livro, armazenando informações básicas como nome, autor
e quantidade de páginas.

### Dados do modelo
- Nome do livro
- Autor
- Número de páginas

### Operações
- Construtor para inicialização dos atributos
- Métodos getters e setters
- Método `muitasPaginas`, que indica se o livro possui mais de 300 páginas

### Conceitos aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Uso de atributos do tipo `String` e `int`
- Retorno de valor booleano
