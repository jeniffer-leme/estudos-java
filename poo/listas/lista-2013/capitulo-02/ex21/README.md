## Exercício 21 – Livro Livraria

Implementação da classe **LivroLivraria**, conforme o modelo apresentado no
exercício 1.20.

### Descrição
A classe representa um livro disponível para venda em uma livraria, contendo
informações sobre o livro e dados relacionados à comercialização.

### Dados do modelo
- Nome do livro
- Autor
- Número de páginas
- Preço
- Quantidade em estoque
- Código identificador

### Operações
- Construtor para inicialização dos atributos
- Métodos getters e setters
- Método `venderLivro`, que controla a venda e atualiza o estoque

### Regras
- A venda só é realizada se houver quantidade suficiente em estoque

### Conceitos aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Controle de estoque
- Retorno de valor booleano
