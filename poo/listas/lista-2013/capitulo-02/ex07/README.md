## Exercício 7 – Lâmpada

Implementação da classe **Lampada**, baseada no modelo apresentado na listagem 1.1 do livro.

### Descrição
A classe representa uma lâmpada simples, que pode estar ligada ou desligada,
utilizando um atributo para controlar seu estado.

### Atributo
- `estadoLampada`: valor booleano que indica se a lâmpada está acesa (`true`)
  ou apagada (`false`).

### Operações
- Construtor para definir o estado inicial
- Métodos para acender e apagar a lâmpada
- Método para consultar o estado atual

### Conceitos aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Uso do tipo boolean
