## Exercício 24 – Data

Implementação da classe **Data**, contendo um método para duplicar os valores
de outra instância da mesma classe.

### Descrição
A classe representa uma data composta por dia, mês e ano, permitindo copiar
os valores de uma outra instância de `Data`.

### Dados do modelo
- Dia
- Mês
- Ano

### Operações
- Construtor para inicialização da data
- Métodos getters e setters
- Método `duplicarData`, que copia os valores de outra data

### Conceitos aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Passagem de objetos como parâmetro
