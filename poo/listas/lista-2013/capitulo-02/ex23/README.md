## Exercício 23 – Contador

Implementação da classe **Contador**, que encapsula um valor utilizado para
contagem de itens ou eventos.

### Descrição
A classe mantém um valor inteiro de contagem que não pode ser acessado
diretamente por programadores-usuários, permitindo apenas operações controladas
por métodos específicos.

### Dados do modelo
- Valor de contagem (inteiro)

### Operações
- Zerar o contador
- Incrementar o contador
- Imprimir o valor atual do contador

### Regras
- O valor do contador é acessado apenas por meio de métodos
- Não é permitido imprimir valores negativos

### Conceitos aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Controle de acesso a atributos
- Uso de exceções
