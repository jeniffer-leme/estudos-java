## Exercício 29 – Conta Bancária Simplificada

Implementação da classe **ContaBancariaSimplificada**, correspondente ao modelo
apresentado na listagem 1.2 do livro.

### Descrição
A classe representa uma conta bancária simples, contendo informações sobre o
titular, saldo e se a conta é especial ou não. O acesso aos dados é controlado
por meio de modificadores de acesso e métodos específicos.

### Dados do modelo
- Nome do titular da conta
- Saldo
- Indicador de conta especial

### Operações
- Abertura de conta
- Depósito de valores
- Retirada de valores com validação de saldo

### Regras
- Contas não especiais não podem ter saldo negativo
- Contas especiais podem realizar retiradas mesmo sem saldo suficiente

### Conceitos aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Modificadores de acesso
- Regras de negócio
- Uso de exceções
