## Exercício 19 – Ponto 2D

Implementação da classe **Ponto2D**, correspondente ao modelo apresentado como
resposta do exercício 1.18.

### Descrição
A classe representa um ponto em um plano cartesiano bidimensional, definido
pelas coordenadas `x` e `y`.

### Dados do modelo
- Coordenada `x` (valor real)
- Coordenada `y` (valor real)

### Operações
- Construtor para inicializar as coordenadas
- Métodos getters e setters
- Método para exibir o ponto no formato `(x, y)`

### Conceitos aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Uso de atributos do tipo `double`
