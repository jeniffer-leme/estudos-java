## Exercício 10 – Hora Aproximada

Criação do modelo **HoraAproximada**, baseado no modelo `Data` (listagem 1.3),
com o objetivo de representar uma hora qualquer do dia utilizando horas e minutos.

### Descrição
A classe armazena valores inteiros para hora e minuto, garantindo que apenas
horários válidos sejam aceitos.

### Dados do modelo
- Hora (0 a 23)
- Minuto (0 a 59)

### Operações
- Construtor com validação dos valores
- Métodos getters e setters
- Validação de horário
- Exibição da hora no formato `hora:minuto`

### Conceitos aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Validação de dados
- Tratamento de exceções
