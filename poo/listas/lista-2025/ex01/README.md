# Exercício 01 - Classe Data

Este exercício implementa uma classe `Data` em Java com os seguintes requisitos:

## Funcionalidades

- Atributos encapsulados: `dia`, `mes` e `ano`.
- Construtor para inicializar os atributos.
- Métodos `set` e `get` para cada atributo.
- Método `mostrarData()` → exibe a data no formato `dia/mes/ano`.
- Método `vemAntes(Data data)` → retorna `true` se a data atual vier antes da data passada como argumento (ou for igual).
- Método `eBissexto()` → retorna `true` se o ano for bissexto.
